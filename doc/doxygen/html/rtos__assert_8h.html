Detailed Description

Implementation of macro ASSERT for the Arduino board. If the assertion fires the code attempts to write an error string into the global Serial object (its initialization therefore is a prerequisite of using ASSERT), wait for a while and than makes a reset.

Implementation of macro ASSERT for the Arduino board. If the assertion fires the code attempts to write an error string into the global Serial object (its initialization therefore is a prerequisite of using ASSERT), wait for a while and than makes a reset.
If the compilation is not made in DEBUG mode ASSERT expands to nothing.
