Define Documentation

#define CLK_TASK_TIME_RTUINOS_STANDARD_TICS   123

To be used to configure the regular task of the real time clock implementation: The task has to be called every CLK_TASK_TIME_RTUINOS_STANDARD_TICS RTuinOS standard system clock tics.

Remarks:
The chosen value doesn't strongly matter. It must not be out of the range of the data type of the system time, i.e. not greater than 127. The greater it is the lower is the overhead of the RTOS, but if it gets to large the update of the display might become visibly irregular. We choose a crude value to demonstrate the don't matter character.

void clk_taskRTC ( void  )

Regular task function for real time clock support.

The regular task function of the real time clock. Has to be called every 100th tic of the RTuinOS system time, which is running in its standard copnfiguration of about 2 ms a tic.
